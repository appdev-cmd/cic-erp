2.2 Chi phí thực hiện HĐ

Danh sách linh hoạt. Mỗi hạng mục có: Tên (gợi ý tự động), % (tính trên tổng đầu vào), Số tiền. Nhập % → tiền tự cập nhật và ngược lại.

2.3 Chiết khấu từ NCC

Nhập % chiết khấu → Hệ thống tính: Chiết khấu = Tổng đầu vào × %. Chiết khấu GIẢM tổng chi phí.

📥 3. Import hàng loạt từ Excel

Các bước thực hiện

  1. Trong danh sách HĐ → nhấn 📥 Import
  2. Nhấn Tải file Template mẫu → tải file Excel mẫu
  3. Điền dữ liệu vào file theo bảng cột bên dưới
  4. Kéo thả file hoặc nhấn Chọn file
  5. Kiểm tra preview: ✅ hợp lệ / ⚠️ lỗi
  6. Nhấn "Import N hợp đồng"

📊 4. Import từ file PAKD

PAKD (Phương Án Kinh Doanh) là file Excel chuẩn nội bộ CIC. Khi import sẽ tự động:

✅ Tạo sản phẩm mới nếu chưa có

✅ Tạo nhà cung cấp mới nếu chưa có

✅ Import chi tiết hạng mục + ngoại tệ

✅ Import chi phí thực hiện HĐ

✅ Tính chiết khấu NCC (tiền → %)

✅ Điền vào form tự động

Cách dùng: Trong form → Bước 2 → nhấn Import PAKD → chọn file .xlsx

⚠️Import PAKD sẽ GHI ĐÈ toàn bộ sản phẩm & chi phí hiện tại trong form.

🧮 5. Công thức tính toán

Giá trị Ký kết = Σ (SL × Giá Đầu ra)
Tổng Đầu vào = Σ (SL × Giá Đầu vào)
Tổng Chi phí = Đầu vào + CP Trực tiếp + CP Thực hiện − Chiết khấu NCC
Lợi nhuận Gộp = Ký kết − Tổng Chi phí
Tỷ suất LN = (LN Gộp / Ký kết) × 100%
Doanh thu (có VAT) = Ký kết / (1 + VAT%)

Ví dụ

Hạng mụcSLGiá vàoGiá raCP TT
Phần mềm Revit58,000,00012,000,0000
Đào tạo BIM25,000,00015,000,00010,000,000
Tư vấn BEP1050,000,0000
Ký kết = 60M + 30M + 50M = 140,000,000
Đầu vào = 40M + 10M + 0 = 50,000,000
CP TT = 10,000,000 | Chiết khấu 5% = 2,500,000
Tổng CP = 50M + 10M − 2.5M = 57,500,000
LN Gộp = 82,500,000 | Tỷ suất = 58.9%
