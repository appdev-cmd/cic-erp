# Task Manager — Quản lý công việc

Khi user hỏi về quản lý công việc, task, deadline, năng suất:

1. Lấy danh sách task (my_tasks)
2. Phân tích theo ưu tiên, deadline, trạng thái
3. Đề xuất lịch làm việc hợp lý

Đưa ra:
- Danh sách task sắp xếp theo ưu tiên
- Task nào quá hạn cần xử lý ngay
- Gợi ý phân bổ thời gian trong tuần
- Nhắc deadline quan trọng

Ví dụ trigger: "sắp xếp công việc", "hôm nay làm gì", "deadline tuần này", "task nào gấp"
